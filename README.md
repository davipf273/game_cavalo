# 🐎 Game Cavalo

> Um jogo simples e divertido onde você controla um cavalo e enfrenta desafios para alcançar a maior pontuação possível.

---

## 📌 Sobre o projeto

O **Game Cavalo** é um projeto desenvolvido com foco em prática de programação, utilizando tecnologias web.
O jogo explora conceitos fundamentais como lógica de jogo, renderização gráfica e interação com o usuário.

Esse projeto foi criado como parte de aprendizado e desenvolvimento de habilidades em programação.

---

## 🎮 Jogabilidade

* Controle o cavalo na tela
* Desvie de obstáculos 
* Sobreviva o máximo possível
* Alcance a maior pontuação

---

## 🎯 Objetivos do projeto

* Praticar **JavaScript** na criação de jogos
* Trabalhar com **canvas ou renderização gráfica**
* Desenvolver lógica de movimentação e colisão
* Criar uma estrutura organizada de código

---

## 🚀 Funcionalidades

* ✅ Movimentação do personagem
* ✅ Sistema de pontuação
* ✅ Detecção de colisão
* ✅ Loop de jogo (game loop)
* ✅ Interface simples e funcional
